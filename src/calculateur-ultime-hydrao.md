## 🧠 LOGIQUE DE CALCUL

### Inputs utilisateur
```typescript
interface CalculatorInput {
  household: 'solo' | 'couple' | 'family' | 'family-plus';
  bottlesPerWeek: number;
  priorities: Array<'savings' | 'health' | 'environment' | 'time'>;
}
```

### Formules de calcul
```typescript
// Économies bouteilles
const avgBottlePrice = 0.50; // Moyenne France
const yearlyBottles = bottlesPerWeek * 52;
const yearlyBottleCost = yearlyBottles * avgBottlePrice;
const monthlyBottleCost = yearlyBottleCost / 12;

// Coût HYDRAO
const hydroPrice = 490;
const subscriptionYearly = 99; // Formule Standard moyenne
const hydroYearlyCost = subscriptionYearly;
const hydroYear1Cost = hydroPrice + subscriptionYearly;

// Économies réelles
const yearlySavings = yearlyBottleCost - hydroYearlyCost;
const savings5Years = (yearlyBottleCost * 5) - (hydroPrice + (subscriptionYearly * 5));

// Break-even
const breakEvenMonths = hydroPrice / monthlyBottleCost;

// Microplastiques (étude récente : 240 000 particules/bouteille)
const microplasticsPerBottle = 240000;
const yearlyMicroplastics = yearlyBottles * microplasticsPerBottle;

// Environnement
const plasticWeightPerBottle = 0.034; // kg (bouteille 50cl)
const yearlyPlasticKg = yearlyBottles * plasticWeightPerBottle;
const co2PerBottle = 0.063; // kg CO2 (production + transport)
const yearlyCO2kg = yearlyBottles * co2PerBottle;
const treesEquivalent = Math.round(yearlyCO2kg / 12); // 1 arbre absorbe ~12kg CO2/an

// Temps
const shoppingTrips = Math.ceil(yearlyBottles / 26); // ~26 bouteilles par course
const hoursPerTrip = 0.75; // 45 min moyenne
const shoppingHours = shoppingTrips * hoursPerTrip;

const kettleBoilsPerDay = household === 'solo' ? 3 : household === 'couple' ? 5 : household === 'family' ? 8 : 12;
const kettleWaitMinutes = 3; // Temps d'attente moyen
const kettleHoursPerYear = (kettleBoilsPerDay * kettleWaitMinutes * 365) / 60;

const totalTimeSaved = shoppingHours + kettleHoursPerYear;

// Cost per day
const costPerDayWithoutHydro = yearlyBottleCost / 365;
const costPerDayWithHydro = hydroYearlyCost / 365; // An 2+
```

## 🧪 VARIANTES À TESTER (A/B tests)

### Test 1 : Ordre des résultats
- **Variante A :** Économies → Santé → Environnement → Temps
- **Variante B :** Santé (choc microplastiques) → Économies → Env → Temps
- **Hypothèse :** Santé en premier = impact émotionnel plus fort

### Test 2 : Affichage "coût de l'inaction"
- **Variante A :** Afficher "Si vous achetez dans 6 mois = 624€ perdus"
- **Variante B :** Ne pas afficher (moins agressif)
- **Hypothèse :** Urgence sans pression = meilleur taux conversion

### Test 3 : Social proof
- **Variante A :** "87% des familles comme vous ont choisi..."
- **Variante B :** "Plus de 15 000 familles équipées"
- **Variante C :** Témoignage client avec photo
