## 8. ANTI-PATTERNS À ÉVITER

### ❌ Ce qui TUE la conversion (à ne JAMAIS faire)

#### 1. Pop-up Exit Intent
```
"ATTENDEZ ! -10% si vous restez"
```
**Pourquoi NON** : Vous êtes premium. Pas un site cheap.

#### 2. Countdown Timer Factice
```
"Plus que 3h pour profiter de l'offre !"
```
**Pourquoi NON** : Détruit la confiance adulte.

#### 3. Social Proof Inventé
```
"Jean de Paris vient d'acheter il y a 2min"
```
**Pourquoi NON** : Détectable, ridicule.

#### 4. Trop de CTAs
```
[Acheter] [Calculer] [Configurer] [Télécharger] [S'inscrire]
```
**Pourquoi NON** : Paralysie du choix.

#### 5. Vidéo Auto-Play avec Son
**Pourquoi NON** : Pire UX ever.
