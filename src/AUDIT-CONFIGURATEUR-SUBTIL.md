## 🎨 PRINCIPES DE DESIGN

### 1. ESPACEMENT GÉNÉREUX
```
Pas de saturation visuelle.
Chaque élément respire.
Blanc = luxe.
```

### 2. TYPOGRAPHIE SOBRE
```
Pas de MAJUSCULES partout.
Pas de gras excessif.
Hiérarchie naturelle.
```

### 3. COULEURS DISCRÈTES
```
#6B1E3E pour guidage subtil (border, petit accent)
Pas de dégradés agressifs
Pas de rouge "URGENCE"
Gris doux pour secondaire
```

### 4. ANIMATIONS DOUCES
```
Transitions 300-400ms
Easing naturel (ease-out)
Pas de bounce ou elastic
Pas de confetti
```

### 5. MICRO-COPY HUMAIN
```
"Pour vous :" (pas "VOICI VOTRE CONFIG")
"Le plus choisi" (pas "🏆 POPULAIRE")
"Voir le détail" (pas "CALCULER VOS ÉCONOMIES >>>")
```
