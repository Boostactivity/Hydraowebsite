## 🎨 DESIGN SYSTEM DES CALCULATEURS

### Composant unifié : `<Calculator>`

```tsx
interface CalculatorProps {
  type: 'discovery' | 'analysis' | 'validation';
  category: 'financial' | 'behavioral' | 'health' | 'environmental' | 'comparative' | 'predictive' | 'social';
  
  inputs: CalculatorInput[];
  outputs: CalculatorOutput[];
  
  visualType?: 'chart' | 'timeline' | 'comparison' | 'badge' | 'score';
  cta?: { text: string; action: () => void };
  
  subtleMode?: boolean; // Mode discret (panier, configurateur)
}

interface CalculatorInput {
  id: string;
  type: 'slider' | 'select' | 'toggle' | 'number';
  label: string;
  defaultValue: any;
  range?: [number, number];
  options?: Array<{ value: string; label: string }>;
}

interface CalculatorOutput {
  id: string;
  label: string;
  value: number | string;
  format: 'currency' | 'percentage' | 'number' | 'duration' | 'weight';
  emphasis?: 'primary' | 'secondary' | 'accent';
  icon?: React.ReactNode;
}
```

## 🔄 FLUX DE DONNÉES : Calculateurs interconnectés

```typescript
// Store centralisé des données utilisateur
interface UserCalculatorData {
  profile: {
    household: 'solo' | 'couple' | 'family' | 'family-plus' | 'pro';
    members: number;
    hasKids: boolean;
    lifestyle: 'sedentary' | 'active' | 'athlete';
  };
  
  consumption: {
    bottlesPerWeek: number;
    avgBottlePrice: number;
    teaCoffeePerDay: number;
    sparklingWater: boolean;
  };
  
  configuration: {
    model: string;
    finish: string;
    capacity: string;
    waterModule: string;
    subscription: string;
  };
  
  location: {
    postalCode?: string;
    city?: string;
  };
}

// Calculateurs partagent ce store
const calculatorStore = createStore<UserCalculatorData>();

// Exemple : Homepage calcule → Page Économies pré-remplit
Homepage: calculatorStore.set('consumption', { bottlesPerWeek: 12 })
↓
Page Économies: inputs pré-remplis avec bottlesPerWeek: 12
↓
Configurateur: affiche "Avec 12 bouteilles/semaine = 1200€/an économisés"
```

## 🎯 RÈGLES D'OR DE LA SUBTILITÉ

1. **Jamais plus de 3 chiffres** affichés simultanément
2. **Toujours un CTA clair** après chaque résultat
3. **Progressive disclosure** : simple d'abord, détails ensuite
4. **Pas de pop-ups** calculateurs (sauf abandon cart)
5. **Résultats en <0,5s** (calcul instantané)
6. **Graphiques minimalistes** (pas de 3D, pas de dégradés agressifs)
7. **Langage humain** : "Vous économisez" > "Économies calculées"
8. **Ancrage subtil** : "vs 1 200€/an en bouteilles" (jamais en gros)
