## 2. ARCHITECTURE DU PARCOURS UTILISATEUR

### 2.1 Les 3 Profils Visiteurs

#### PROFIL A : "Le Coup de Cœur" (40% du trafic)
**Comportement** :
- Visite < 2min
- Scroll rapide
- Décision émotionnelle

**Besoin** :
- Désir instantané
- Preuve sociale immédiate
- CTA simple et clair

**Parcours idéal** :
```
Homepage (Hero désir) 
  → Découverte Simple (5 usages + message humain)
  → Configurateur (Personnalisation = engagement)
  → Achat Direct
```

---

#### PROFIL B : "Le Rationnel" (35% du trafic)
**Comportement** :
- Visite 5-10min
- Compare, calcule, vérifie
- Décision après justification

**Besoin** :
- Données chiffrées
- Comparaison concurrents
- Preuves techniques

**Parcours idéal** :
```
Homepage (Hero désir)
  → Prix Transparent (Ancrage 6240€ → 490€)
  → Calculateur Économies (Personnalisé)
  → Sécurité (Rassurance)
  → Achat Réfléchi
```

---

#### PROFIL C : "Le Sceptique" (25% du trafic)
**Comportement** :
- Visite 10-20min
- Cherche les défauts
- Lit les petites lignes

**Besoin** :
- Témoignages vidéo
- Garanties détaillées
- Installation expliquée

**Parcours idéal** :
```
Homepage (Hero désir)
  → Témoignages (Social proof fort)
  → Sécurité (Page exemplaire à garder)
  → Installation (Vidéo 30s)
  → FAQ (Objections levées)
  → Garantie (Risk reversal)
  → Achat Confiant
```
